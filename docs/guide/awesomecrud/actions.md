# Actions

The crud compoenent allows to define which actions are enabled on a component. There are many default actions that can
be configured.

- advancedFiltering
- autoRefresh
- bulkDelete
- bulkEdit
- changeDisplayMode
- changeDisplayMode // Show's the controls for switching between table, list and kanban
- changeItemsPerRow
- collapse
- collapse
- columnsFilters
- create
- dateFilter
- delete
- edit
- editLayout
- export
- filter
- formPagination
- formPagination
- import
- noActions
- pagination
- refresh
- search
- tableConfiguration
- view
